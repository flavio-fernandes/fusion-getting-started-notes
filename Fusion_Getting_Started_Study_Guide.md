# Autodesk Fusion -- Getting Started Series -- Searchable Study Guide

This is a searchable, timestamped index for the 16-video Autodesk Fusion Getting Started series.

It is generated from the transcripts (`transcripts/*.txt`) and the reference notes PDFs (`resources/*.pdf`).

Note: PDFs are present for videos 3-5 and 7-16. Videos 1, 2, and 6 do not have a PDF in `resources/`.

## How to use this guide

- If you know the feature name, jump to **Concept index** and click a timestamp.
- If you know the video number, jump to **Master index** and click the section link.
- If you only remember what you wanted to do, try searching this page for a keyword:
  - `constraint`, `dimension`, `timeline`, `revolve`, `extrude`, `sweep`, `loft`, `form`, `joint`, `interference`, `drawing`, `toolpath`, `configuration`

## Glossary

Short definitions for the jargon used across the series.

- **3D sketch** -- A sketch mode that allows sketch entities that are not confined to a single plane, useful for paths used in Sweep.
- **Appearance** -- Visual materials, colors, and finishes applied to bodies and components (rendering and visual organization).
- **As-built joint** -- A joint created between components that are already positioned, so Fusion does not move them while defining the relationship. Useful for assemblies that are already in place.
- **Associativity** -- When changes in one place propagate automatically, such as changes in a model updating drawings and CAM.
- **Auto Constrain** -- A sketch option that automatically applies constraints while you sketch. Great for speed, but you should still verify constraints and dimensions so the sketch is fully defined.
- **Body** -- A solid or surface piece of geometry. Bodies can live inside components.
- **Browser** -- The left-side tree that shows components, bodies, sketches, construction geometry, and document settings.
- **Canvas** -- An inserted reference image (often for tracing) inside a sketch or design.
- **Centerline / Construction line** -- A sketch line marked as construction. Commonly used as an axis of symmetry or revolve axis, and it does not create a profile by itself.
- **Chamfer** -- A beveled edge, usually defined by distance and/or angle.
- **Coincident constraint** -- Forces a point to lie on another point, line, or curve.
- **Collinear constraint** -- Forces two lines to lie on the same infinite line, making them perfectly aligned.
- **Component** -- A container for geometry and features. Components help keep assemblies organized and make joints, motion, and reuse much easier.
- **Configuration** -- A table-driven way to generate variations of a design by controlling parameters and feature suppression.
- **Constraint** -- A rule that locks sketch geometry behavior, like horizontal, vertical, coincident, tangent, equal, collinear, etc.
- **Construction geometry** -- Reference-only geometry (construction lines, axes, planes) used for alignment and modeling. It does not create solid material.
- **Data panel** -- The file and project browser (cloud storage) inside Fusion. Used for upload, open, version history, sharing, and team hubs.
- **Dimension** -- A numeric value that drives size or position in a sketch. Dimensions are the core of parametric editing.
- **Direct modeling / Direct edit** -- Edits made directly to faces and geometry (move face, press-pull), often without changing the original feature definitions.
- **Drawing (documentation)** -- 2D documentation output: views, dimensions, parts lists, and sheets.
- **Edit in Place** -- Edit a linked or inserted component in-context while seeing the full assembly, then return to the top-level design.
- **Equal constraint** -- Forces two lines (or arcs) to have the same length (or radius).
- **Extend to adjacent faces (Form)** -- Extends a form feature so it blends or connects into neighboring faces.
- **Extrude** -- Turns a sketch profile into a 3D feature (solid or cut) by pushing it linearly.
- **Extrude to Object** -- An extrude option that ends the extrusion when it hits a selected face or body, instead of a fixed distance.
- **Feature suppression** -- Disables a feature without deleting it. Often used for configurations or troubleshooting.
- **Fill hole (Form)** -- A Form command that closes an open boundary in a T-spline body.
- **Fillet** -- A rounded edge, usually defined by a radius.
- **Form (Sculpt)** -- Freeform modeling environment based on T-splines. Great for organic shapes and smooth transitions.
- **Guide rail** -- A curve used by Loft to control shape between profiles.
- **Hole feature** -- A parametric hole tool that supports simple, counterbore, countersink, threading, and placement by sketch points.
- **Home icon** -- UI control that returns you to the Fusion home view (project list).
- **Insert edge (Form)** -- Adds extra edges to a T-spline form to increase control for shaping.
- **Interference** -- An inspection that finds overlapping volume between components or bodies in an assembly.
- **Joint** -- Defines how components relate and move (rigid, revolute, slider, etc.).
- **Linked design** -- A referenced design file that stays connected, allowing updates to propagate.
- **Loft** -- Creates a surface or solid that transitions through multiple profiles. Often used with guide rails for control.
- **Marking menu** -- The right-click context menu (radial) for quick access to common commands.
- **Mirror** -- Creates symmetric geometry or features across a plane or line.
- **Motion link** -- Links the motion of two joints so they move together with a ratio, like gears or coupled rotations.
- **Offset** -- Creates a parallel sketch curve at a given distance, or offsets a face/surface.
- **Origin** -- The base reference point and axes (X, Y, Z) of the design or component.
- **Parameters** -- Named values (user parameters or model parameters) that can drive dimensions and features across the design.
- **Parts list / BOM** -- A table in drawings listing components, quantities, and metadata.
- **Pattern** -- Repeats features or bodies in a circular or rectangular arrangement.
- **Plane along a path** -- A construction plane created perpendicular to a curve at a specified point along that curve.
- **Post-process** -- Converts toolpaths into machine-specific G-code using a post processor.
- **Press Pull** -- A direct edit tool that pushes or pulls faces and can also create offsets or extrudes depending on selection.
- **Profile (Sketch profile)** -- A closed region in a sketch that can be used by features like Extrude, Revolve, and Loft.
- **Project (Sketch)** -- Projects existing edges or geometry into a sketch so you can reference it with constraints and dimensions.
- **Public link** -- A shareable link to a Fusion design or project, typically read-only unless permissions are granted.
- **Rail (Loft)** -- A guide curve used to control loft shape between profiles.
- **Revolve** -- Creates a 3D feature by spinning a sketch profile around an axis.
- **Rigid group** -- Locks multiple components together so they behave as one rigid unit in assembly motion.
- **Section analysis** -- A visual cut-through inspection tool to view interior details and check clearances.
- **Setup (Manufacture)** -- Defines the stock, work coordinate system, and orientation for manufacturing operations.
- **Simulation (CAM)** -- Preview of toolpaths to validate cutting moves and check for collisions.
- **Sketch** -- 2D geometry used to drive 3D features. Typically built on a plane and controlled with constraints and dimensions.
- **Sketch palette** -- The sketch-side panel that contains toggles and options like Show Constraints, Auto Constrain, and visibility controls.
- **Smart dimension / Auto-dimension** -- Tools that automatically suggest or apply drawing dimensions based on geometry and common drafting rules.
- **Suppression** -- Turning a feature on or off (often for configurations).
- **Sweep** -- Creates geometry by moving a profile along a path (a curve).
- **Takeoff (Loft)** -- A loft control that influences the direction the loft leaves a profile.
- **Takeoff / Weight (Loft)** -- Loft controls that influence how the loft transitions from a profile and how strongly a guide rail influences the shape.
- **Tangent constraint** -- Forces a line/arc to touch another curve smoothly without a corner.
- **Team hub** -- A shared Fusion hub for collaboration across users and projects.
- **Timeline** -- The feature history at the bottom. You can roll back, edit features, and reorder where allowed.
- **Toolpath** -- A CAM path that defines how a machine tool moves to cut or drill material.
- **Weight (Loft)** -- A loft control that adjusts how strongly a guide rail pulls the loft shape.
